Возможности и преимущества приложения

  • Angular

    Проект выполнен с использованием JavaScript фреймворка Angular

  • Система каталога и поиска товаров

    В приложении есть каталог с делением на категории и подкатегории, а также поле ввода для поиска

  • Профиль пользователя

    Пользователь может зарегистрироваться и войти в свой профиль

  • Корзина, избранное и заказы

    Пользователь может добавлять товары в корзину, список любимых товаров и делать заказ

  • Панель сортировки

    Сортировки на странице каталога позволяют находить наилучший товар

  • Адаптивность

    Приложение адаптировано под дисплеи до 500px
